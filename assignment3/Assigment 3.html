Assignment 3

In this assignment you will implement the SimpleList class using a linked list as the data structure.  The assignment will be done in 3 phases.

Phase 1:

Download SimpleListL.h.  This is the file we went over in class.  Write code in SimpleList.cpp that implements the constructor, empty, listSize, append and display methods.  Write a test program to verify that these methods work.  In lab on Friday (Feb. 19) you will be provided a test program to run with your SimpleListL implementation.  To earn full credit for the lab your SimpleListL implementation must pass this test.

Phase 2:

Add implementations for the retrieve, insert and remove methods.  Write a program that thoroughly tests these methods.  You may start work on this before or in lab if you want to.  It is highly recommended that you try to complete the retrieve and the insert methods in lab.

Phase 3:

Add the big 3 methods (destructor, copy constructor and operator=).  Also add an iterator and iterator methods (these will be discussed in class on Thursday).  Write your own test program to make sure that all these methods work.

Assignment Submission:

Submission to Blackboard is due by 10 pm on Wednesday Feb. 24. The .tar.gz file that you submit to Blackboard must contain only SimpleListL.h and SimpleListL.cpp.  Your class will be tested by a program provided by the instructors.

Submit hard copy of SimpleListL.h and SimpleListL.cpp at the beginning of class on Thursday Feb. 25.
