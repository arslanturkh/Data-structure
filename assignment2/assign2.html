Assignment 2

For this assignment you will need to download SimpleList.h, SimpleList.cpp and players.txt and put them in the same directory as player.h and player.cpp. Change the itemtype defined in SimpleList.h from int to the name of the class you defined in player.h.  Make no other changes to the 3 downloaded files.

Write a program that:

  • creates a SimpleList object which contains items of type TennisPlayer. The file players.txt contains the first name, last name, point total and country of the top 20 ranked players in order of their ranking. Notice that some country names are 2 words, for ex. Great Britain.

  • Allows the user to make the following queries:

    • Which player is ranked at a given position?

      • ex: who is number 4?

      • position given must be in range 1..20

    • What is the ranking of a player with a given last name?

      • ex: what is the ranking of Nadal?

      • the answer could be that the player is not ranked.

    • How many points separate the players in 2 specified positions?

      • ex: how many points separate the players ranked 3 and 5?

      • positions must be in range 1..20

Your program will consist of 3.cpp files (player, SimpleList and the program) and 2 .h files (player and SimpleList). Modify the makefile you wrote for lab2 so it automates the compiling and linking of these files to create the executable program.

In addition to the criteria used to grade Assignment 1 your program will be graded on how well you use functions to decompose the program. The function main should consist primarily of calls to other functions, each of which handles a piece of the job to be done. A well decomposed program to solve this problem will have a minimum of 6 or 7 functions (in addition to main).
