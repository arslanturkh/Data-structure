Assignment 1



Write a C++ program that prompts the user for a first name, last name and age and then displays this information in the format: last name, first name, age

Your program must be decomposed into 4 functions:

  • main – which calls the other 3 functions

  • a function that prompts the user for a first name and a last name

    • whatever the user enters should be accepted as valid

  • a function that prompts the user for an age

    • you can assume that the user enters an integer value but must keep prompting until a value between 0 and 125 is entered

  • a function that displays the information in the specified format

You must use each of the following communication paths at least once:

  • a value parameter

  • a reference parameter

  • a const reference parameter

  • a return value

Your program must be organized as shown in class and you must following the style standards discussed in class. Also, you must use C++ style strings and I/O.

Hard copy submission is due at the beginning of class on: Thursday Feb. 4. In addition to the source code include output produced by 2 executions of the program (i.e. test cases).
